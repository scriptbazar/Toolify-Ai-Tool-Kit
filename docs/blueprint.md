# **App Name**: ToolifyAI

## Core Features:

- Smart Utility Tools: Collection of 100+ smart utility tools categorized by type (text, PDF, image, SEO, dev, productivity).
- AI Tool Suite: AI-powered tool tool suite, including AI writer, resume builder, chatbot, PDF Q&A, YouTube summarizer, code helper, and image generator.
- Categorized Tool Display: Display of the tools with clearly organized categories: text, PDF, AI, dev, image, SEO, embed tools
- Adsense Monetization: Google AdSense integration with auto-slot support on free tools and pages.
- AI Prompt Templates: AI tool prompt templating system. Manage prompts used to generate LLM responses, customize for each AI tool. Make it configurable via the admin panel. Prompt content can be modified directly to tune AI responses.

## Style Guidelines:

- Primary color: Deep violet (#673AB7) to represent intelligence, creativity, and digital innovation.
- Background color: Light grey (#F5F5F5), nearly white, to ensure a clean, modern, and easily readable interface. 
- Accent color: Electric purple (#DF00FF), a vivid, eye-catching color to highlight calls to action and important interactive elements.
- Body and headline font: 'Inter' sans-serif for a clean, modern, objective look.
- Use a set of modern, minimalist icons in a consistent style across all tools.
- Responsive grid-based layout that adapts to different screen sizes.
